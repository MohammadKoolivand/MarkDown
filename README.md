## 1.Heading
You can use one to six `#` sign for headings. So below codes

```
#heading 1

##heading 2

###heading 3

####heading 4

#####heading 5

######heading 6
```

will ends to 

# heading 1
## heading 2
### heading 3
#### heading 4
##### heading 5
###### heading 6

## 2. Quote
We have to type of quoting. 

### Quoting Text
You can quote with `>` sign

`> This is my Quote`

> This is my Quote

### Quoting Code
Fo r quoting code you should just put the code between two backticks.

```
git status
git add
git commit
```
